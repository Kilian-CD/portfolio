# Portfolio de Kilian

Bienvenue sur mon portfolio en ligne ! 🌐

Ce projet présente mes compétences, mes projets et mon expérience en développement web et design. Il est principalement réalisé avec HTML, CSS et JavaScript.

---

## 🚀 Démo

Vous pouvez consulter le site en direct ici : [https://kiliancd.fr](https://kiliancd.fr)

---

## 🛠️ Installation / Utilisation

Si vous souhaitez cloner ou contribuer au projet :

```
# Cloner le dépôt
git clone https://github.com/Kilian-CD/portfolio.git

# Se rendre dans le dossier du projet
cd portfolio

# Ouvrir le fichier index.html dans votre navigateur
```

> Ce projet ne nécessite aucun serveur particulier, il peut être ouvert directement dans un navigateur.

---

## ✨ Fonctionnalités

* Présentation des projets personnels
* Compétences et expériences détaillées
* Animations et effets interactifs (glow sur les SVG, transitions, etc.)
* Responsive design pour mobile et desktop
